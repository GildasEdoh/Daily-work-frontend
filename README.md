# Daily Work

Ce projet est une application web construite avec Next.js, TypeScript et Tailwind CSS. Elle permet la gestion et la visualisation de rapports, l'analyse de données, et la gestion d'utilisateurs.

## Fonctionnalités principales
- Authentification et gestion des utilisateurs
- Tableau de bord analytique
- Création, modification et visualisation de rapports
- Filtres et recherche avancée
- Interface moderne et responsive

## Structure du projet
- `app/` : Pages et routes de l'application
- `components/` : Composants réutilisables (UI, layout, analytics, reports)
- `hooks/` : Hooks personnalisés
- `lib/` : Utilitaires, types, et données mock
- `public/` : Fichiers statiques
- `store/` : Gestion d'état (stores)
- `styles/` : Feuilles de style globales

## Installation
1. Clonez le dépôt :
   ```bash
   git clone <url-du-repo>
   ```
2. Installez les dépendances :
   ```bash
   pnpm install
   ```
3. Lancez le serveur de développement :
   ```bash
   pnpm dev
   ```

## Configuration
- Modifiez les fichiers de configuration (`next.config.mjs`, `tsconfig.json`, etc.) selon vos besoins.
- Ajoutez vos variables d'environnement dans `.env.local`.

## Technologies utilisées
- Next.js

- Tailwind CSS
